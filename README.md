Teo Node.js
==========

Run Teo server and write custom URL route handlers with Node.js.

## Installation

```sh
npm install oink-format
```

## Example

Write with JavaScript:

```javascript
const { App } = require("oink-format")

const app = new App()
app.run()

```

Write with TypeScript:

```typescript
import { App } from "oink-format"

const app = new App()
app.run()

```
